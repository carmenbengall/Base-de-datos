# Módulo de Bases de Datos

Este repositorio contiene los ejercicios y scripts realizados en el módulo de **Bases de Datos**.  
Incluye actividades de teoría.

## Contenido
- **Actividades:** actividades de teoría realizadas en clase.
- **Proyectos:** ejercicios de teoría.

## Objetivo
Aprender a diseñar, crear y manipular bases de datos relacionales usando SQL, esta es la teoría.
